# fighting-console (콘솔 격투기 게임)

## 소개

- **fighting-console**은 콘솔 기반의 간단한 턴제 격투 게임입니다.
- 플레이어와 AI가 교대로 공격하며, 주사위로 선공자를 결정합니다.
- 다양한 액션과 랜덤성으로 승패가 갈리는 재미있는 미니게임입니다.

## 게임 방법

1. 프로그램을 실행합니다.
2. 플레이어 이름을 입력하고 AI와 대전이 시작됩니다.
3. 선공자는 주사위로 결정하며, 각 턴마다 플레이어와 AI가 번갈아 공격합니다.
4. 공격은 [주먹, 발차기, 어퍼컷, 로우킥] 중 선택 입력합니다.
   - 잘못 입력하면 기본 공격(주먹)으로 자동 처리됩니다.
5. HP(체력)가 먼저 0이 되는 쪽이 패배합니다.
6. AI는 무작위로 공격을 선택합니다.

## 주요 클래스 설명

- `Main.java` : 게임 시작 진입점
- `game.GameManager` : 게임 전체 흐름 제어(플레이어/AI 생성, 턴/승패 처리)
- `game.ConsoleUI` : 콘솔 출력 및 안내 메시지 담당
- `fighter.PlayerFighter` : 플레이어 캐릭터 (키보드 입력)
- `fighter.AIFighter` : AI 캐릭터 (랜덤 행동)
- `fighter.Fighter` : 격투가 공통 추상 클래스 (이름, HP, 행동)
- `action.Action` : 공격(액션) 종류, 실행 로직
- `game.Dice` : 주사위(랜덤) 유틸리티

## 실행 방법

```bash
javac -d out/production/fighting-game src/**/*.java
java -cp out/production/fighting-game Main
```
